# I.D. Guide

**I.D. Guide** is a free, community-driven resource designed to help people in Canada navigate legal name, gender marker, and identity document changes. The site provides clear, step-by-step guides, official forms, and practical resources to support users in updating their documents with confidence and ease. The project especially aims to support trans, non-binary, Two-Spirit, and gender non-conforming communities.


## Features
- Step-by-step guides for legal name changes, gender marker updates, birth certificates, health cards, driver's licenses, passports, permanent resident cards, SIN, CRA, and more
- Downloadable official forms and applications (English and French)
- Region-specific guides for Alberta, Manitoba, and Ontario
- Canada-wide resources and support organizations
- Workshop and presentation materials
- Accessibility-first design: skip links, keyboard navigation, semantic markup
- Community feedback and support channels

## Getting Started
This project is built with [Next.js](https://nextjs.org/) using React, TypeScript, CSS Modules, and modern web standards.

To run locally:

```bash
git clone https://github.com/torontopubliclibra/idguide.git
cd idguide
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Accessibility
I.D. Guide is committed to accessibility, featuring skip links, focusable navigation, and semantic markup to ensure a usable experience for all visitors. If you need information in a different format, please [contact us](mailto:contact@idguide.ca).
