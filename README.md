# I.D. Guide

**I.D. Guide** is a free, community-driven resource designed to help Canadians navigate legal name and gender marker changes. The site offers clear, step-by-step guides, official forms, and practical resources to support users in updating their identity documents with confidence and ease.

## Features

- Comprehensive guides for updating Canadian identity documents (name, gender marker, health card, passport, and more)
- Downloadable official forms and applications
- Workshop and presentation resources
- Accessibility-first design, including skip links and keyboard navigation
- Community feedback and support channels

## Getting Started

This project is built with [Next.js](https://nextjs.org/) using React, CSS Modules, and modern web standards.

To run locally:

```bash
git clone https://github.com/torontopubliclibra/idguide.git
cd tg-id
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Accessibility

I.D. Guide is committed to accessibility, featuring skip links, focusable navigation, and semantic markup to ensure a usable experience for all visitors.
