
# I.D. Guide

**I.D. Guide** is a free, community-driven resource for navigating legal name and gender marker changes in Canada. The site provides clear, step-by-step guides, official forms, and practical resources to help people update their identity documents with confidence.

## Features

- Step-by-step guides for Canadian document changes
- Downloadable official forms and applications
- Workshop and presentation resources
- Accessibility-focused design
- Community feedback and support

## Getting Started

This project is built with [Next.js](https://nextjs.org/) and uses React, CSS Modules, and modern web standards. To run locally:

```bash
git clone https://github.com/torontopubliclibra/tg-id.git
cd tg-id
npm install
npm run dev
```

Then visit [http://localhost:3000](http://localhost:3000) in your browser.
